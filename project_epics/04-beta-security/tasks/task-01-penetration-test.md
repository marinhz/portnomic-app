# Task 4.1 — Penetration test and remediation

**Epic:** [04-beta-security](../epic.md)

---

## Objective

Execute external or internal penetration test; remediate critical and high findings (EDD §12).

## Scope

- Engage pen test (external or internal); scope: auth, API, tenant isolation, file/PDF access, admin endpoints.
- Document findings; prioritize critical/high; fix or accept with mitigation plan.
- Re-test or spot-check after remediation.
- Align with threat model: unauthorized access, data leakage, AI manipulation, financial fraud, abuse/DoS (EDD §12).

## Acceptance criteria

- [ ] Pen test completed; report received and reviewed.
- [ ] Critical and high findings remediated or explicitly accepted with documented mitigation.
- [ ] No known critical/high open issues before beta/production sign-off.
